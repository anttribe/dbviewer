# DbViewer - 关系型数据库查看工具

## 介绍
DbViewer：实现在web中查看数据库的详细信息，主要功能：1.数据源信息维护； 2.数据库信息展示（树结构）；3.数据库结构查看：表（ER图）、视图、索引等；4.数据库文档生成（开发中）

## 数据库支持

- [x] MySQL 

## 文档生成支持（开发中）
- [x] html
- [x] word
- [x] markdown
